This repository contains all materials, code examples, sections, and projects for the **Computer Graphics Course (CSX51)** using **OpenGL** and **C++** at the Faculty of Computers and Artificial Intelligence Benha University.

---

## 📋 Contents

### Sections
- **2D Animation**
- **Coloring**
- **2D Transformations**
- **Keyboard & Mouse Events**
- **3D Drawing and Transformations**
- **Lighting**
- **Texture Mapping**
- **Collision Detection**

---

## 📚 Topics Covered

### 2D Animation
- Double Buffering
- Front Buffer & Back Buffer
- `glutSwapBuffers()`
- `glutTimerFunc()`
- `glutPostRedisplay()`

### Coloring
- RGB & RGBA Coloring
- `glClearColor()`
- `glColor3f()`
- Smooth & Flat Shading

### 2D Transformations
- Translation, Rotation, and Scaling
- Matrix Transformations
- `glTranslatef()`, `glRotatef()`, `glScalef()`

### Keyboard & Mouse Events
- Keyboard Interaction
- Special Keys Handling
- Mouse Events
- Fullscreen Mode
- Interactive Object Movement

### 3D Graphics
- Orthographic Projection
- Perspective Projection
- `gluPerspective()`
- `gluLookAt()`
- Depth Buffer (`GL_DEPTH_TEST`)

### Lighting
- Diffuse Reflection
- Specular Reflection
- Ambient Lighting
- Light Sources (`GL_LIGHT0`, ...)
- Material Properties

### Texture Mapping
- Texture Coordinates
- Texture Wrapping (Repeat, Clamp, Mirror)
- Texture Filtering (Nearest, Linear, Mipmap)
- Texture Generation

### Collision Detection
- Circle Collision Detection
- Axis-Aligned Bounding Box (AABB)
